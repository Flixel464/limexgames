Development Tools


The tools that we use and recommend for game development!



Visual Studio Code

Visual studio code is a great tool for programming as it has a nicely designed interface and is capable with being used with roblox via Rojo.

Rojo

The software that allows us to use Visual Studio Code.

Yucon Framework

A great game framework that helps us keep our games organized, optimized and secure.

(This is only usable on Visual Studio Code as long as you use our edited rojo project for it found at https://github.com/Flixel464/Yucon-Framework-Rojo)

Blender

Blender is a great 3d modelling tool that we use for all of our game projects, whether it is making trees or creating new and unique weapons.

paint.net

paint.net is a great photo editing and drawing tool that we use to create 2d assets for our game.
